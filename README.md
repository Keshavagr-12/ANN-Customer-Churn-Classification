# ANN-Customer-Churn-Classification
Customer Churn Prediction using Artificial Neural Networks (ANN), TensorFlow, and Streamlit.

## Project Overview
This project predicts whether a customer will leave the bank using an Artificial Neural Network (ANN).

## Features
- Data preprocessing
- Label Encoding
- One-Hot Encoding
- Feature Scaling
- ANN Model using TensorFlow/Keras
- Streamlit Web Application
- TensorBoard Visualization

## Technologies Used
- Python
- TensorFlow
- Scikit-learn
- Pandas
- NumPy
- Streamlit

## Dataset
Churn Modelling Dataset
